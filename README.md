# Delivery-Comida-Vegana
Sistema de pedidos usando POO, DDD e TDD em Python.
Loja Virtual - Comida Vegana Sim! -> in PROGRESS!!


Estrutura
Sistema de Pedidos

- Customer
    name
    email

- Product
    name
    description
    price

- Order
    [OrderStatus]
    [OrderItem]
    total

- OrderSttus
    name (REALIZADO, EM PREPARAÇÂO, ENVIADO, ENTREGUE, FINALIZADO)

- OrderItem // objeto de valor
    - product_id
    - price
    - quantity
